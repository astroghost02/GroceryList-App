# GroceryList-App

The Grocery WebApp streamlines grocery list management with features like creating, organizing, and printing lists, sorting items by category, searching for products, and managing quantities. Administrators can update product details, manage categories, and oversee user accounts through a control panel. The app integrates with databases like MySQL for data storage and uses real-time updates via RESTful APIs.
